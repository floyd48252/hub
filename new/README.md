# Industrial & Domestic Cleaning Solutions

A professional cleaning services management system optimized for Chrome.

## 🚀 Quick Start

### For Chrome Users:
1. Run `start-server.bat` (Windows) or start the server manually
2. The script will launch your browser to `http://localhost:8080/`
3. You'll see the home page directly - no directory listing!
4. The site is now fully optimized for Chrome

## 🌐 Chrome Features

- **Progressive Web App (PWA)**: Install as a desktop app
- **Offline Support**: Service worker caches essential files
- **Responsive Design**: Optimized for all Chrome screen sizes
- **Modern UI**: Gradient backgrounds and smooth animations
- **Local Storage**: Data persists between sessions
- **No Directory Listing**: Root URL serves home page directly

## 📱 PWA Installation

1. Open the site in Chrome at `http://localhost:8080/`
2. Click the install icon in the address bar (or menu)
3. The app will be added to your desktop/start menu

## 🔗 Direct Access URLs

- **Home**: `http://localhost:8080/` (auto-redirects to home)
- **Login**: `http://localhost:8080/login.html`
- **Register**: `http://localhost:8080/register.html`
- **Admin Login**: `http://localhost:8080/admin-login.html`
- **Customer Dashboard**: `http://localhost:8080/customer-dashboard.html`
- **Admin Dashboard**: `http://localhost:8080/admin-dashboard.html`

## 🔧 Technical Details

- **Server**: Python HTTP server on port 8080
- **Storage**: Browser localStorage (Chrome-compatible)
- **Manifest**: PWA manifest for app-like experience
- **Service Worker**: Offline caching and background sync
